## Overview
A shopping cart application for an online plant shop offering aromatic and medicinal house plants.

## Features
- Landing page with navigation
- Product listing with plant cards
- Cart with quantity controls, delete, and subtotal
- Dynamic cart icon showing item count
- Checkout and continue shopping buttons

## Tech Stack
- React + Vite
- Redux Toolkit
- CSS for styling

git clone https://github.com/<username>/e-plantShopping.git
cd e-plantShopping
